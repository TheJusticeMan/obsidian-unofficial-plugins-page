Obsidian Unofficial Plugins Registry

This registry is automatically updated weekly from the Obsidian releases repository.

Data Files

  • installable-plugins.json
    Plugins containing a valid manifest.json and main.js.
  • installable-themes.json
    Themes containing a valid manifest.json and theme.css.
  • registry-complete.json
    Complete registry data including all PRs, files, and manifests.
  • registry-files.json
    Registry data including file lists but excluding manifests.
  • registry-index.json
    Basic index of PRs and repositories.
